# Text Sentiment Classifier using Naive Bayes

## Description
This project implements two versions of Naive Bayes classifiers for sentiment analysis:
- Multinomial Naive Bayes (based on word frequency counts)
- Bernoulli Naive Bayes (based on binary word presence)

## Requirements
- Python 3.6+
- scikit-learn (for evaluation metrics only)

## Installation
1. Extract the zip file
2. Ensure you have the data file `data.tsv` in the same directory
3. Install required packages: `pip install scikit-learn`

## Usage
Run the main script:
```bash
python main.py
